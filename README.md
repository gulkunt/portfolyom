# Kişisel Portföy Web Sitesi

Bu proje, kişisel portföyümü tanıtmak amacıyla geliştirilmiş bir web sitesidir. 
Proje kapsamında HTML ve CSS kullanılarak responsive (uyumlu) bir tasarım oluşturulmuştur.

## Özellikler
- Kişisel bilgiler bölümü
- Hakkımda sayfası
- İletişim bölümü
- Responsive tasarım
- Basit ve kullanıcı dostu arayüz

## Kullanılan Teknolojiler
- HTML5
- CSS3

## Amaç
Bu proje, web geliştirme becerilerimi geliştirmek ve projelerimi sergilemek amacıyla hazırlanmıştır.
